# MAMS Frontend Demo

Αυτό είναι το πρώτο demo για το Medication Adherence and Monitoring System (MAMS).

- Ξεκίνησα την υλοποίηση από το βασικό feature: **Medication Reminder**.
- Έφτιαξα ένα απλό Angular UI με:
  - Κουμπί **Send Ping** για δοκιμή.
  - Εμφάνιση τελευταίου **medication reminder**.
- Το demo τρέχει στο **StackBlitz** λόγω εταιρικού περιορισμού (δεν μπορώ να κάνω npm install τοπικά).
- Ο κώδικας είναι ανεβασμένος στο GitHub.

- Ανοίξτε το project στο StackBlitz: (https://stackblitz.com/edit/stackblitz-starters-f5shbezk?file=src%2Fmain.ts)

## Τι περιέχει
- **app.component.ts**: Το UI με κουμπί και εμφάνιση δεδομένων.
- **socket.service.ts**: Mock service που στέλνει ping και reminders.
- **main.ts**: Εκκίνηση της εφαρμογής.
- **index.html** και **global_styles.css**: Βασικά αρχεία.
